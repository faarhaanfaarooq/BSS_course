# Lists

## About

This folder covers different types of HTML lists used to structure content in web pages.

## Files

- **index.html** - Demonstrates different types of HTML lists

## Content

This file showcases four types of HTML lists:

1. **Unordered List (`<ul>`)** - Bulleted list items using `<li>` tags
2. **Ordered List (`<ol>`)** - Numbered list items (note: each `<li>` should be a child, not separate `<ol>` tags)
3. **Nested List** - Lists within lists, combining unordered lists
4. **Definition List (`<dl>`)** - Key-value pairs with `<dt>` (term) and `<dd>` (description) tags
