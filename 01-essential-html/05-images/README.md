# Images

## About

This folder covers HTML image elements, including different ways to specify image sources and accessibility attributes.

## Files

- **index.html** - Demonstrates different image linking techniques

## Content

This file showcases:

1. **Relative Paths** - Images from local folders (`images/landscape.jpg`)
2. **Absolute URLs** - Images from external sources (using placeholder services)
3. **Width and Height** - Setting dimensions directly in HTML
4. **Alt Text** - Alternative text for accessibility and SEO
5. **Title Attribute** - Tooltip on hover
6. **Figure and Figcaption** - Semantic way to add captions to images

## Key Attributes

- `src` - Image source (path or URL)
- `alt` - Alternative text (required for accessibility)
- `width` / `height` - Image dimensions
- `title` - Tooltip text
