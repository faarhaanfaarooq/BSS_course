# Anchor Tags

## About

This folder covers HTML anchor tags (`<a>`) used for creating hyperlinks to navigate between pages, sections, external sites, and more.

## Files

- **index.html** - Demonstrates different types of anchor links

## Content

This file showcases various types of links:

1. **External Links** - Links to other websites
2. **Same Tab vs New Tab** - Using `target="_blank"` to open in new tab
3. **Relative Links** - Links to files in the same directory (e.g., `about.html`)
4. **Internal Links** - Links to sections within the same page using IDs
5. **Email Links** - Using `mailto:` protocol to open email client
6. **File Links** - Links to downloadable files (PDFs, documents)
7. **Title Attribute** - Adding tooltip text with `title` attribute

## Key Attributes

- `href` - The destination URL
- `target` - Where to open the link (`_self`, `_blank`)
- `title` - Tooltip text on hover
