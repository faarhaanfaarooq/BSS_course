# Line Breaks, Horizontal Rules & HTML Entities

## About

This folder covers HTML formatting elements and special character entities used in web development.

## Files

- **index.html** - Demonstrates HTML formatting elements and character entities

## Content

This file covers:

1. **Line Breaks (`<br>`)** - Forces text onto a new line
2. **Horizontal Rules (`<hr>`)** - Creates a visual divider between sections
3. **Preformatted Text (`<pre>`)** - Preserves whitespace and formatting
4. **Code Display (`<code>`)** - Displays inline code snippets
5. **HTML Entities** - Special characters that need escaping:
   - `&lt;` - Less than (<)
   - `&gt;` - Greater than (>)
   - `&amp;` - Ampersand (&)
   - `&nbsp;` - Non-breaking space
   - `&copy;` - Copyright symbol (©)
   - `&reg;` - Registered trademark (®)
   - `&trade;` - Trademark (™)
   - `&deg;` - Degree symbol (°)
