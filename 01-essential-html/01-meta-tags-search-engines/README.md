# HTML & CSS - Meta Tags

This project demonstrates the use of **meta tags** in an HTML document. Meta tags provide metadata about the webpage, which is not displayed directly on the page but is essential for search engines, browsers, and accessibility tools.

## Meta Tags Used

### 1. `<meta charset="UTF-8">`
- **Purpose:** Defines the character encoding for the document.
- **Why:** Ensures that text (including special characters and symbols) is displayed correctly across different browsers and devices.

### 2. `<meta name="viewport" content="width=device-width, initial-scale=1.0">`
- **Purpose:** Controls how the webpage is displayed on mobile devices.
- **Why:** Makes the site responsive by scaling content to fit the screen size, improving mobile usability.

### 3. `<meta name="description" content="Learn HTML & CSS from this cool Course">`
- **Purpose:** Provides a short summary of the webpage.
- **Why:** Search engines often use this description in search results, which can improve click-through rates and SEO.

### 4. `<meta name="keywords" content="HTML, CSS, Web Development">`
- **Purpose:** Lists keywords relevant to the page content.
- **Why:** Historically used by search engines to understand page topics. While modern search engines rely less on this tag, it can still provide context.

## Why Meta Tags Are Important

- 🖥️ **Browser Behavior:** Helps browsers interpret and render the page correctly.
- 📱 **Responsive Design:** Ensures the site adapts to different screen sizes and devices.
- 🔍 **SEO (Search Engine Optimization):** Improves visibility in search results by providing descriptions and context.
- 🌍 **Accessibility & Usability:** Assists tools like screen readers and translation services in understanding the page content.

---

## Example Output

When you open the page, the browser uses these meta tags to:
- Display characters correctly (UTF-8).
- Scale the layout properly on mobile devices.
- Show a meaningful description in search results.
- Provide context about the page’s topic.
