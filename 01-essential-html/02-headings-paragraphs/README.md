# Headings & Paragraphs - Lesson 2

This file demonstrates the fundamental building blocks of HTML content.

## What This File Covers

### 1. HTML Document Structure
- `<!DOCTYPE html>` - Declares the document type (HTML5)
- `<html lang="en">` - Root element with language attribute
- `<head>` - Contains metadata (charset, viewport, title)
- `<body>` - Contains visible content

### 2. Headings (`<h1>` to `<h6>`)
Six levels of headings establish document hierarchy:
- `<h1>` - Main title (one per page)
- `<h2>` to `<h6>` - Subheadings in descending importance

### 3. Paragraphs (`<p>`)
Block-level containers for text content.

### 4. Inline Text Formatting
- `<strong>` - Bold for important text
- `<em>` - Italics for emphasized text
- `<mark>` - Highlighted text
- `<del>` - Strikethrough (deleted)
- `<ins>` - Underline (inserted)
- `<sub>` - Subscript
- `<sup>` - Superscript

## Why This Matters

1. **Semantic HTML** - Headings communicate document structure to browsers and screen readers
2. **SEO** - Search engines use headings to understand content hierarchy
3. **Accessibility** - Screen readers rely on semantic elements for navigation
4. **Readability** - Proper headings and paragraphs make content scannable
5. **Typography** - Inline formatting adds meaning (emphasis, scientific notation like H₂O or E=mc²)
